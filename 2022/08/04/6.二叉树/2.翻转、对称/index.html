针对二叉树的问题，解题之前一定要想清楚究竟是前中后序遍历，还是层序遍历。

思考：

  1. 前序遍历和后序遍历均可实现；
  2. 中序遍历的递归会导致某些节点被翻转两次，需作修改。（靠指针来遍历的）
  3. 而中序遍历的迭代则不用修改，因为是靠栈来遍历的。（很有意思）
  4. 层序遍历也可以实现。

572.另一个树的子树

给你两棵二叉树 root 和 subRoot 。检验 root 中是否包含和 subRoot 具有相同结构和节点值的子树。如果存在，返回 true ；否则，返回 false 。

二叉树 tree 的一棵子树包括 tree 的某个节点和这个节点的所有后代节点。tree 也可以看做它自身的一棵子树。

示例；

输入：root = [3,4,5,1,2], subRoot = [4,1,2]

输出：true

输入：root = [3,4,5,1,2,null,null,null,null,0], subRoot = [4,1,2]

输出：false

  • 思路1：dfs暴力匹配

一个树是另一个树的子树则：要么这两个树相等、要么这个树是左树的子树、要么这个树是右树的子树。

  • 思路2：KMP

若一个树是另一个树的子树，则它们前序遍历的结果一定是包含关系；

但反过来讲不一定，例如2分别是1的左/右子树

通过添加两个起标记作用的空节点lNull、rNull来完善前序遍历。是否匹配：KMP
