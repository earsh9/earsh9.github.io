693. 交替位二进制数

给定一个正整数，检查它的二进制表示是否总是 0、1 交替出现：换句话说，就是二进制表示中相邻两位的数字永不相同。

  • 逐位检查

  • 利用交替二进制数的特点：

    • 若 n 为交替二进制数，则其右移一位得到的值 m 仍为交替二进制数

    • 将 n 与 m 异或会得到形如 00011…11 的结果 x

    • 将 x 加一（进位后）得到形如 00100.。0 的结果，将该结果与 x 按位与 会得到全 0 的结果

762. 二进制表示中质数个计算置位

给定一个整数区间 [left,right]，计算区间内所有数字二进制表示中 1 的个数，若个数为素数，计数加一，返回最终结果。

1 <= left <= right <= 10^6

0 <= right - left <= 10^4

  • 知识点回顾：

    • 求整数 x 的二进制有多少个 1:

      • x &= (x - 1) -> 消除最低位的 1，while 循环 x != 0

      • 用 lowBit 实现：x & (-x) -> 只保留整数 x 最低位的 1

    • 如何判断一个数为素数

      • for(int i = 2; i * i <= x; i++) if(x % i == 0) return false;

    • 巧用 mask，数据最多有20位，使用一个数 mask 存储所有质数信息：若 i 为质数则置 第 i 位为 1

      • 对于任何一个数的 bitCount, 若 (1 << bitCount & mask) != 0 -> 该 bitCount 为质数
