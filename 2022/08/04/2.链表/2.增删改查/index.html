19题 删除链表的倒数第n个节点

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

进阶：你能尝试使用一趟扫描实现吗？

设置快慢指针，让快指针先走n步再一起走，快指针走到头时，慢指针刚好走到待删的节点。

有几个细节：

  1. ListNode dummy = new ListNode(-1, head);

  2. ListNode slow = dummy,fast = dummy;

  3. while(n– && fast != null) fast = fast.next; 先走n步

  4. fast = fast.next; 再多走一步

    fast指针要再多走一步即共走了n + 1步，如此slow 才能走到待删节点之前。

    链表长度是一定的，fast多走一步，slow就少走一步。
