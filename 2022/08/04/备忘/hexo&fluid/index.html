*.github.io的个人博客网站无法连接的问题

如果 hexo 要部署上 Github，对应的 Github 仓库需以 xxx.github.io 命名，其中 xxx 为 Github 注册时的名字。

如果是项目名字错误了可以直接在github上项目的setting那里修改就可以了，不需要删掉重新建。

更新部署步骤

更换主题后，本地服务器运行成功，部署到github上就不能显示主题样式

  • 禁用缓存
  • hexo clean之后 hexo deploy -g
  • 删.deploy_git文件夹，然后hexo d
  • GitHub 删掉仓库重新部署（注意默认分支是啥 _config.yml 里就写啥分支）
